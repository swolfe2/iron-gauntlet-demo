# Iron Gauntlet: Rust Warden (Demo)

**A high-stakes reaction combat game built with Phaser 3 & TypeScript.** 
Master directional blocking and 'Just Frame' parries to defeat the Rust Warden. Features mobile-first design, procedural audio, and skill-based boss mechanics.
